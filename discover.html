Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answar: The properties innerHTML, innerText, and textContent are used to get or modify the content inside an HTML element.

Question-3: What is event delegation in the DOM?

Answar: Event delegation is a technique in JavaScript where a single event listener is added to a parent element to handle events on multiple child elements, even if they are dynamically added later.

Question-4: What is event bubbling in the DOM?

Answar: Event bubbling is a phase of event propagation in the DOM where an event starts from the target element and propagates (bubbles) up through its ancestors (parent elements) until it reaches the root.

Question-5: How do you create, add, and remove elements using JavaScript?

Answar: To create a new element in JavaScript, use the document.createElement() method. There are several ways to insert an element into the DOM like appendChild(), prepend(), append() etc. To remove an element, use one of these methods like remove().
